APOLLO MCP SERVER — TOOL REFERENCE (44 tools)

══════════════════════════════════════════════════
 SEARCH (4 tools)
══════════════════════════════════════════════════

1. search_people
   Search Apollo's 270M+ contact database with precision filters.
   Build targeted prospect lists by job title, seniority, location,
   company size, industry, tech stack, and more. Returns profile data
   but NOT emails/phones — use enrich_person for contact details.
   ─────────────────────────────────────────────
   Params:
     person_titles          string[]  optional  Job titles (OR logic). e.g. ['CFO', 'VP Finance']
     q_keywords             string    optional  Keyword search across name, title, company, bio
     person_locations       string[]  optional  e.g. ['California, US', 'New York, US']
     person_seniorities     string[]  optional  'founder','c_suite','vp','director','manager','senior','entry'
     contact_email_status   string[]  optional  'verified', 'guessed', 'unavailable'
     q_organization_domains string    optional  Company domains (newline-separated)
     organization_locations string[]  optional  Company HQ locations
     organization_ids       string[]  optional  Specific Apollo organization IDs
     organization_num_employees_ranges string[] optional  e.g. ['1,10', '51,200']
     prospected_by_current_team string[] optional  ['yes'] = saved, ['no'] = net new
     page                   number    optional  Default 1
     per_page               number    optional  1-100, default 25

2. search_organizations
   Search Apollo's company database to find target accounts. Filter
   by employee count, HQ location, keywords, and more.
   ─────────────────────────────────────────────
   Params:
     q_organization_name    string    optional  Company name to search
     q_organization_keyword_tags string[] optional  e.g. ['ecommerce', 'saas', 'fintech']
     organization_num_employees_ranges string[] optional  e.g. ['51,200', '201,500']
     organization_locations string[]  optional  HQ locations
     organization_not_locations string[] optional  Exclude HQ locations
     organization_ids       string[]  optional  Specific Apollo org IDs
     prospected_by_current_team string[] optional  ['yes'] = in CRM, ['no'] = new
     page                   number    optional  Default 1
     per_page               number    optional  1-100, default 25

3. get_organization
   Get complete details for a specific organization by Apollo ID.
   Returns full profile including funding, tech stack, headcount.
   ─────────────────────────────────────────────
   Params:
     organization_id        string    required  Apollo organization ID

4. get_organization_jobs
   View current job postings for a company. Useful for understanding
   hiring priorities and pain points before outreach.
   ─────────────────────────────────────────────
   Params:
     organization_id        string    required  Apollo organization ID

 ENRICHMENT (4 tools)
══════════════════════════════════════════════════

5. enrich_person
   Look up detailed info about a person — email, phone, title,
   company, employment history. Provide any combo of name, email,
   LinkedIn URL, company domain to match.
   ─────────────────────────────────────────────
   Params:
     first_name             string    optional  Person's first name
     last_name              string    optional  Person's last name
     name                   string    optional  Full name (alt to first/last)
     email                  string    optional  Email address
     organization_name      string    optional  Company name
     domain                 string    optional  Company domain (e.g. 'acme.com')
     linkedin_url           string    optional  LinkedIn profile URL
     reveal_personal_emails boolean   optional  Set true to include personal emails
     reveal_phone_number    boolean   optional  Set true to include phone numbers (1 mobile credit)

6. bulk_enrich_people
   Enrich up to 10 people simultaneously. Efficient for batch
   enrichment of prospect lists.
   ─────────────────────────────────────────────
   Params:
     people                 object[]  required  Array of person identifiers (max 10)
       ├─ first_name        string    optional
       ├─ last_name         string    optional
       ├─ name              string    optional
       ├─ email             string    optional
       ├─ organization_name string    optional
       ├─ domain            string    optional
       └─ linkedin_url      string    optional
     reveal_personal_emails boolean   optional  Set true to include personal emails
     reveal_phone_number    boolean   optional  Set true to include phone numbers

7. enrich_organization
   Get comprehensive company data by domain — industry, employee
   count, revenue, funding, tech stack, LinkedIn, and more.
   ─────────────────────────────────────────────
   Params:
     domain                 string    required  Company domain (e.g. 'shopify.com')

8. bulk_enrich_organizations
   Enrich up to 10 companies simultaneously by domain.
   ─────────────────────────────────────────────
   Params:
     domains                string[]  required  Array of domains (max 10)

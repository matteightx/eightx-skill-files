CONTACTS (18 tools)
================================================================================

1. contacts_search
   Search contacts with filters. Find contacts by name, email, phone, tags,
   custom fields, date ranges, and more. Returns paginated results.
   Params:
     - query (string, optional): Search query string (name, email, phone)
     - filters (array, optional): Array of filter conditions
         field (string): Field to filter on
         operator (string): eq, neq, contains, starts_with, ends_with, gt, lt, gte, lte, exists, not_exists, between
         value (string|number|boolean|array): Filter value
     - page (number, optional): Page number (starts at 1)
     - pageLimit (number, optional): Results per page (max 100, default 20)
     - sortBy (string, optional): Field to sort by
     - sortOrder ("asc"|"desc", optional): Sort order

2. contacts_get
   Get a single contact by ID. Returns full contact details including custom
   fields, tags, and attribution data.
   Params:
     - contactId (string, required): The contact ID

3. contacts_create
   Create a new contact. Provide at least name or email. Supports custom fields,
   tags, and DND settings.
   Params:
     - firstName (string, optional): First name
     - lastName (string, optional): Last name
     - name (string, optional): Full name (alternative to first+last)
     - email (string, optional): Email address
     - phone (string, optional): Phone number (E.164 format preferred)
     - companyName (string, optional): Company name
     - website (string, optional): Website URL
     - tags (array of strings, optional): Tags to assign
     - source (string, optional): Lead source
     - address1 (string, optional): Street address
     - city (string, optional): City
     - state (string, optional): State
     - postalCode (string, optional): Postal/ZIP code
     - country (string, optional): Country
     - timezone (string, optional): Timezone (e.g., America/New_York)
     - gender (string, optional): Gender
     - dateOfBirth (string, optional): Date of birth (YYYY-MM-DD)
     - assignedTo (string, optional): User ID to assign contact to
     - customFields (array, optional): Custom field values [{id, field_value}]
     - dnd (boolean, optional): Do Not Disturb flag
     - dndSettings (object, optional): DND settings per channel

4. contacts_update
   Update an existing contact. Only include fields you want to change.
   Params:
     - contactId (string, required): The contact ID to update
     - firstName, lastName, name, email, phone, companyName, website, tags,
       source, address1, city, state, postalCode, country, timezone,
       assignedTo, customFields, dnd (all optional, same as create)

5. contacts_upsert
   Create or update a contact. If matching contact found (by email or phone),
   it updates. Otherwise creates new.
   Params:
     - firstName, lastName, name, email, phone, companyName, tags, source,
       assignedTo, customFields (all optional)

6. contacts_delete

   Params:

7. contacts_find_duplicates
